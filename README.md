# waya-plugins

A free [Claude Code](https://claude.com/claude-code) plugin marketplace from [Waya](https://github.com/wayaframes) — skills and tools for founders and operators.

## Install the marketplace

```sh
/plugin marketplace add wayaframes/waya-plugins
```

Then browse and install plugins:

```sh
/plugin install <plugin-name>@waya-plugins
```

## Available plugins

| Plugin | For | Skills |
| --- | --- | --- |
| [`pre-pmf-starter`](./plugins/pre-pmf-starter) | Pre-PMF founders | `personal-ikigai`, `pre-pmf-executive-summary`, `mission-values-guide`, `okr-workflow-pre-pmf` |
| [`expert-onboarding`](./plugins/expert-onboarding) | New Waya expert network members | `personal-ikigai` |
