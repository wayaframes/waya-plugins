# expert-onboarding

A Claude Code plugin for **new members joining the Waya expert network**.

## What's inside

1. **`personal-ikigai`** — discover personal alignment with the work you're stepping into. (Same skill content as in `pre-pmf-starter`, framed for experts rather than founders.)

## Install

